# Projeto-microcontroladores-scanner-3d
Desenvolvimento de um scanner 3D com microcontroladores para digitalização de objetos físicos. O sistema utiliza sensores, motores e processamento embarcado para captar dados espaciais e gerar modelos 3D aplicáveis em impressão 3D, prototipagem.
